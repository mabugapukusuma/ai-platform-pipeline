# 🚀 AI Platform Pipeline

A multi-stage AI pipeline that converts natural language input into structured application schemas with validation and automatic repair.

---

## 🔥 Features

- 🔹 Intent Extraction from user input  
- 🔹 System Design generation  
- 🔹 Schema Generation (UI, API, Database, Auth)  
- 🔹 Validation using Pydantic  
- 🔹 Automatic Error Repair  
- 🔹 Handles ambiguous and edge-case inputs  

---

## 🧠 Pipeline Flow
User Input → Intent → Design → Schema → Validation → Repair → Final Output

---

## 💻 Tech Stack

- Python  
- Streamlit  
- Pydantic
